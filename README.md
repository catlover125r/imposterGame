# imposterGame

A **web-based party game hub** hosting two games designed for groups playing on their phones.

Play it live: [catlover125r.github.io/imposterGame](https://catlover125r.github.io/imposterGame)

## Games

### Imposter
A social deduction game. Crewmates all share a secret word — the imposter must blend in without knowing it. Works in pass-the-phone mode (one device) or online via Firebase (each player on their own device).

- Custom word lists (saved in Firebase for multiplayer sessions)
- 4-character room codes for joining

### Brasket Handom
Chaotic physics-based basketball — a port of BasketRandom with a retro CRT/neon aesthetic.

- Local or online multiplayer via Firebase input relay
- Box2D physics engine

## Tech Stack

- Vanilla JavaScript, HTML5, CSS3
- **Firebase Realtime Database** — multiplayer sync and room management
- **Box2D physics engine** — basketball simulation
- Google Fonts ("Press Start 2P" retro font)
